FeatherweightJava source file parsing. This package is populated with code generated from the JFlex and java_cup tools. The methods used from this package are:

  • {@link fj.parse.Lexer#Lexer(InputStream)}
  • {@link fj.parse.Parser#Parser(java_cup.runtime.Scanner)}
  • {@link fj.parse.Parser#parse()}
These are all used in {@link fj.main.Main} and are not needed in the typechecker and evaluator.
