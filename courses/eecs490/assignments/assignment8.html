Problem 1 (10 Points)

Recall the verfication conditions for some simple commands discussed in class:

VC(skip, B) = B
VC(c1; c2, B) = VC(c1, VC(c2, B))
VC(x:=e, B) = B[x→e]
VC(if b then c1 else c2, B) = (b ⇒ VC(c1, B)) ∧ (¬b ⇒ VC(c2, B))
VC(whileI b do c, B) = I ∧ (∀ x1..xn. I ⇒ (b ⇒ VC(c, I) ∧ ¬b ⇒ B)), where x1..xn are variables modified in c


Find an appropriate loop invariant and prove the partial correctness assertion for the following program, using the verification condition rules discussed in class shown above.

{a = 1 ∧ b = i} while (b != 0) do (b := b − 1; a := 2 × a) {a = 2i}
